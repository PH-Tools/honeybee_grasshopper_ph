# honeybee_grasshopper_ph/ — the Grasshopper package (UI shims + binaries)

The canvas-facing half of each component: the thin GHPython wrappers and the compiled user objects users install.

## Contents

- `src/` — the GHPython **wrapper** scripts, one `HBPH - <Name>.py` per component. Each sets `ghenv.Component.Name`, builds `IGH`, instantiates the matching `GHCompo_<Name>` worker from `../honeybee_ph_rhino/gh_compo_io/`, and assigns outputs. Also holds `__HBPH__Util_Update_GHCompos.py`, the on-canvas util that regenerates the `.ghuser` + `.py`.
- `user_objects/` — the compiled `.ghuser` binaries (the installable components). **Regenerated inside Grasshopper**, not editable here.

## Notes
- Wrappers are thin — logic lives in the workers under `../honeybee_ph_rhino/gh_compo_io/`.
- Editing flow and the `.ghuser` regeneration round-trip: `../context/ARCHITECTURE.md`.
